1. Introducción y Contexto

Sobre Kairós

Kairós es una Organización No Gubernamental (ONG) dedicada a proporcionar ayuda en distintas causas comunitarias tales como:

  • Ayuda a niños en riesgo de exclusión social
  • Distribución de alimentos y apoyo alimentario
  • Asistencia integral a refugiados y migrantes
  • Programas educativos y de integración
  • Asistencia sanitaria y apoyo psicosocial

El Significado de Kairós

Kairós (καιρός) es un término del idioma griego antiguo que significa "tiempo oportuno" o "el momento preciso". A diferencia de "Chronos" (χρόνος), que se refiere al tiempo cronológico, Kairós representa el instante perfecto, el momento adecuado para actuar.

Concepto clave: Kairós refleja la filosofía fundamental de la organización: llegar en el momento preciso cuando la ayuda es más necesaria, con la acción correcta en el tiempo correcto.

Misión y Objetivos del Proyecto Web

El sitio web de Kairós tiene como propósito:

  • Informar: Comunicar la labor humanitaria de la organización
  • Visibilizar: Mostrar proyectos activos y sus impactos reales
  • Conectar: Facilitar el contacto con personas interesadas en colaborar
  • Concienciar: Crear conciencia sobre necesidades humanitarias en España
  • Inspirar: Promover la participación ciudadana y voluntariado
  • Transmitir valores: Comunicar humanidad, compromiso y esperanza

2. Identidad Visual y Branding

El Logo de Kairós

El logo de Kairós es una composición geométrica que representa dos elementos fundamentales:

  • La letra K: Inicial del nombre de la organización
  • Un reloj de arena abstracto: Simboliza el tiempo, la urgencia humanitaria y el momento oportuno

Simbolismo visual: El reloj de arena representa cómo el tiempo es limitado para las personas en necesidad, y Kairós actúa precisamente cuando más se necesita. Las formas en espejo de la K crean un efecto visual de simetría y equilibrio.

Variantes del Logo

El logo de Kairós se presenta en tres variantes de color, cada una con un propósito específico:

Variante Rosa Magenta (#E02F8E)

Uso: Contextos emocionales y cercanos • Fondos claros • Elementos destacados • Humanidad

Variante Azul Oscuro (#002771)

Uso: Contextos institucionales • Fondos claros y oscuros • Logos principales • Confianza

Variante Amarillo (#FFDE59)

Uso: Contextos optimistas • Fondos oscuros • Elementos de esperanza • Destaque

Normas de Uso del Logo

✓ CORRECTO:
  • Mantener proporciones originales
  • Dejar espacio de respiro mínimo alrededor
  • Usar colores oficiales exactos
  • Aplicar en fondos que proporcionen contraste
✗ INCORRECTO:
  • Distorsionar las proporciones
  • Rotar o invertir el logo
  • Modificar colores sin autorización
  • Añadir efectos no autorizados
  • Usar versiones de baja resolución

4. Tipografía y Jerarquía Visual

Familia Tipográfica 1: Chillax (Logotipo)

Propósito: Exclusivamente para el logotipo de Kairós

Características:
  • Moderna y distintiva
  • Alta legibilidad incluso a tamaños pequeños
  • Personalidad fuerte y memorable
  • Diferenciación visual clara

Importante: La tipografía Chillax NO debe usarse en textos de contenido. Su uso está reservado únicamente para la identidad del logotipo.

Propósito: Todo el contenido y textos del sitio web

Características:

5. Elementos Visuales y Diseño

Formas Abstractas Decorativas

El diseño de Kairós incorpora formas geométricas abstractas (asteriscos/cruces +) como elementos decorativos que se distribuyen en fondos y secciones.

Características de las Formas:

  • Forma base: Asterisco o cruz (+)
  • Tamaños: Variados (desde 20px hasta 150px)
  • Distribución: Orgánica, no en cuadrícula rígida
  • Opacidad: Variable (50% a 100%)
  • Rotación: Libre, sin restricciones

Colores de Formas Decorativas:

#E02F8E
Humanidad
#002771
Compromiso
#FFDE59
Esperanza

Significado de Formas Abstractas:

  • Diversidad de causas y proyectos
  • Multiplicidad de personas ayudadas
  • Conexiones y red de apoyo comunitario
  • Dinamismo y energía de la organización
  • Multiplicidad de objetivos y enfoques

Iconografía

Estilo de Iconos:

  • Tipo: Line icons (iconos de línea)
  • Trazo: 2-3px de grosor
  • Esquinas: Redondeadas
  • Estilo: Minimalista y limpio

Colores de Iconos:

  • Principal: #002771 (Azul)
  • Destacado: #E02F8E (Magenta)
  • Optimista: #FFDE59 (Amarillo)

Categorías de Iconos:

Categoría Uso
Personas y Comunidad Voluntariado, equipo, social
Alimentación Bancos de alimentos, nutrición
Vivienda y Refugio Emergencias, alojamiento
Educación Programas educativos, formación
Salud Asistencia sanitaria, bienestar
Justicia Social Derechos, igualdad

Fotografía

Estilo Fotográfico:

  • Tipo: Imágenes reales de proyectos, no stock abstracto
  • Composición: Personas en primer plano, acción visible
  • Luz: Natural preferentemente, cálida
  • Emociones: Expresiones genuinas, no posadas

Tratamiento de Imágenes:

  • Sin filtros excesivos o artificiales
  • Contraste medio-alto para legibilidad
  • Saturación natural o ligeramente aumentada
  • Composición clara y directa

Valores a Transmitir:

  • Cercanía y humanidad
  • Solidaridad real
  • Esperanza y futuro positivo
  • Dignidad de las personas
  • Acción concreta y visible

8. Guía de Implementación

Requisitos Previos

  • Navegador web moderno (Chrome, Firefox, Safari, Edge)
  • Editor de código (VS Code recomendado)

Opciones de Instalación

Opción 1: VS Code con Live Server

  1. Instalar Visual Studio Code
  2. Instalar extensión "Live Server"
  3. Abrir carpeta del proyecto
  4. Click derecho en index.html
  5. Seleccionar "Open with Live Server"

Opción 2: Abrir Directamente

  1. Navegar a carpeta del proyecto
  2. Doble click en index.html
  3. El navegador abrirá el sitio automáticamente

Configuración Inicial

1. Tipografías Personalizadas

  • Crear carpeta fonts/ en raíz del proyecto
  • Copiar archivos de familia Satoshi
  • Verificar rutas en fuentes.css
  • Instalar tipografía Chillax para logotipo

2. Logo de Kairós

  • Crear carpeta assets/logo/
  • Descargar logos en sus tres variantes
  • Actualizar src en todos los HTML
  • Proporcionar en PNG y SVG

3. Imágenes Locales

  • Crear carpeta img/ en raíz
  • Descargar imágenes externas
  • Optimizar para web (WebP recomendado)
  • Actualizar rutas en HTML

Responsive Design - Breakpoints

/* Desktop (por defecto) */
                    @media (min-width: 801px) { }

                    /* Tablet */
                    @media (max-width: 800px) {
                    .header-container { flex-direction: column; }
                    .valores-container { grid-template-columns: repeat(2, 1fr); }
                    }

                    /* Mobile */
                    @media (max-width: 480px) {
                    .valores-container { grid-template-columns: 1fr; }
                    }

Testing y Validación

  • Validar HTML con W3C Validator
  • Validar CSS con W3C CSS Validator
  • Testear en navegadores principales
  • Verificar responsive en móviles
  • Revisar accesibilidad (WCAG AA)
  • Comprobar velocidad de carga
  • Revisar enlaces (sin broken links)

    • 9. Guía de Mantenimiento

    Actualización de Contenido

    Añadir Nuevo Proyecto

    1. Abrir proyectos.html
    2. Duplicar estructura article.proyecto-card
    3. Actualizar:
      • Título (H2)
      • Descripción (P)
      • Imagen (IMG src y alt)
    4. Guardar y probar en navegador

    Modificar Valores

    1. Abrir quienes-somos.html
    2. Localizar .valores-container
    3. Editar .valor-item:
      • Imagen del ícono
      • Texto del valor
    4. Mantener estructura de 4 columnas

    Actualizar Galería

    1. Abrir galeria.html
    2. En .galeria, añadir/modificar img tags
    3. Mantener 6 imágenes para balance visual

    Cambios de Estilo

    Modificar Colores Corporativos

    /* En css/style.css */
                    :root {
                    --color-primary: #002771;        /* Azul oficial */
                    --color-accent-magenta: #E02F8E; /* Rosa oficial */
                    --color-accent-yellow: #FFDE59;  /* Amarillo oficial */
                    }

    Ajustar Tipografía

    body {
                    font-family: 'Satoshi-Regular', 'Segoe UI', Arial, sans-serif;
                    font-size: 16px;
                    line-height: 1.8;
                    }

                    h1, h2, h3 {

                    }

    Meta Tags
